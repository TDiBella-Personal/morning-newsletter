# Star Battles

A Star Battle puzzle game — a logic puzzle played on a grid divided into regions.

## Rules

1. Place stars in the grid so that each row, column, and region contains exactly the required number of stars.
2. Stars cannot touch each other, not even diagonally.

## Project Structure

```
star-battles/
├── README.md
├── index.html       # Main game page

```

## Getting Started

Open `index.html` in a browser to play.
